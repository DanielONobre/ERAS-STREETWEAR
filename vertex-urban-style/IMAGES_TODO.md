## 🔧 COMO CONVERTER PARA WEBP

```bash
# Instalar cwebp (macOS com Homebrew):
brew install webp

# Converter um arquivo:
cwebp -q 82 foto.jpg -o camiseta-asfalto-02-001.webp

# Converter todos os JPG em lote:
for f in *.jpg; do cwebp -q 82 "$f" -o "${f%.jpg}.webp"; done
```

Qualidade recomendada: **82** (produtos) / **78** (hero) / **80** (lookbook).
